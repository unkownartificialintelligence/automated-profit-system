# 🚀 DEPLOY JERZII AI LANDING PAGE TO VERCEL

## STEP-BY-STEP GUIDE (10 Minutes)

---

### ✅ STEP 1: Create Vercel Account (2 minutes)

1. Go to: https://vercel.com/signup
2. Click "Continue with GitHub" (or Email)
3. Complete signup
4. Verify your email if needed

---

### ✅ STEP 2: Install Vercel CLI (3 minutes)

**Open PowerShell and run:**
```powershell
npm install -g vercel
```

Wait for installation to complete...

---

### ✅ STEP 3: Login to Vercel (1 minute)

**In PowerShell, run:**
```powershell
vercel login
```

**Follow the prompts:**
- Choose your login method (GitHub recommended)
- Browser will open - authorize Vercel
- Return to PowerShell when done

---

### ✅ STEP 4: Deploy Landing Page (2 minutes)

**Make sure you're in the landing-page directory:**
```powershell
cd C:\Users\jerzi\automated-profit-system\automated-profit-system\landing-page
```

**Deploy with one command:**
```powershell
vercel --prod
```

**Answer the prompts:**
- "Set up and deploy?" → **Y** (Yes)
- "Which scope?" → Choose your account
- "Link to existing project?" → **N** (No)
- "What's your project's name?" → **jerzii-ai** (or your choice)
- "In which directory?" → **.** (current directory)
- "Want to modify settings?" → **N** (No)

**🎉 Your site will be LIVE in seconds!**

---

### ✅ STEP 5: Get Your Live URL (Instant)

Vercel will show you:
```
✅ Deployed to production!
🔗 https://jerzii-ai.vercel.app
```

**Copy that URL - Your landing page is NOW LIVE!** 🚀

### 🌐 OPTIONAL: Custom Domain (5-10 minutes)

#### Option A: Use Vercel's Free Domain
Your site is already at: `https://jerzii-ai.vercel.app`
This is professional and works great!

#### Option B: Buy Custom Domain (Recommended)

**1. Buy domain ($10-15/year):**
   - Go to: https://namecheap.com or https://domains.google
   - Search: "jerzii.ai" or "jerzii-ai.com"
   - Purchase the domain

**2. Connect to Vercel:**
   - Go to: https://vercel.com/dashboard
   - Click your project "jerzii-ai"
   - Click "Domains" tab
   - Click "Add Domain"
   - Enter your domain: "jerzii.ai"
   - Follow DNS instructions

**3. Update DNS (at Namecheap/Google):**
   - Add A Record: `76.76.21.21`
   - Add CNAME: `cname.vercel-dns.com`

**Your site will be live at your custom domain in 10-60 minutes!**

## 🚨 TROUBLESHOOTING

### If Vercel CLI doesn't work:
1. Try web interface: https://vercel.com/new
2. Drag and drop your `landing-page` folder
3. Click Deploy

### If deployment fails:
- Make sure `index.html` is in the folder
- Check file encoding (should be UTF-8)
- Contact: support@vercel.com
