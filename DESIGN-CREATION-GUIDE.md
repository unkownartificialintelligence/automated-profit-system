**Cost:** $15/month for ChatGPT Plus

**Step-by-step:**

1. Go to [chat.openai.com](https://chat.openai.com)


## üé® METHOD 2: Canva (Recommended for Beginners)

**Cost:** Free (or $13/month for Canva Pro)

### Setup (One-time):

1. Go to [canva.com](https://canva.com)
2. Sign up for free account
3. Download Canva Pro 30-day trial (optional but recommended)

### Creating Your First Design:

**For "Dog Mom" T-Shirt:**

1. **Create Custom Size**
   - Click "Create a design"
   - Custom size: 4500 x 5400 pixels
   - Click "Create design"

2. **Add Main Text**
   - Click "Text" in left sidebar
   - Add heading
   - Type: "DOG" (press Enter) "MOM"
   - Font: "Fredoka One" or "Baloo 2"
   - Size: 400-500
   - Color: #FF6B6B (coral pink)
   - Align: Center

3. **Add Subtitle**
   - Add another text box
   - Type: "PROUD MEMBER"
   - Font: Same as main text or complementary
   - Size: 150-200
   - Color: #4ECDC4 (turquoise)
   - Position below main text

4. **Add Elements**
   - Click "Elements" in left sidebar
   - Search "paw print"
   - Add 4-6 paw prints around text
   - Color them: #FF6B6B or #4ECDC4
   - Search "heart" and add 2-3 hearts
   - Arrange decoratively

5. **Final Touches**
   - Ensure everything is centered
   - Check spacing and balance
   - Add subtle drop shadow if needed (Effects > Shadow)

6. **Download**
   - Click "Share" > "Download"
   - File type: PNG
   - Check "Transparent background"
   - Click "Download"

**Time: 10-15 minutes for first design, 5 minutes for subsequent**

## üéØ Design Best Practices

### Typography Rules:

1. **Readability First**
   - Test at thumbnail size (can you read it?)
   - Avoid script fonts for main text
   - Use maximum 2-3 fonts

2. **Contrast is King**
   - Dark text on light shirts
   - Light text on dark shirts
   - High contrast = better sales

3. **Size Matters**
   - Main text: 300-600 pts
   - Subtitle: 100-200 pts
   - Leave breathing room

### Color Psychology:

- **Pets:** Warm colors (coral, pink, orange) = love/playful
- **Lifestyle:** Muted pastels = calm/aesthetic
- **Funny:** Bright, bold = energy/humor
- **Holiday:** Red/green/gold = festive
- **Fitness:** Black/red = power/energy

### Layout Tips:

1. **Rule of Thirds**
   - Divide design into 3 sections
   - Place key elements at intersections

2. **White Space**
   - Don't fill every inch
   - Let design breathe
   - 20-30% white space is good

3. **Balance**
   - Symmetrical = professional
   - Asymmetrical = modern/dynamic
   - Choose one approach

### Common Mistakes to Avoid:

‚ùå Using copyrighted images (Disney, logos, etc.)
‚ùå Too much text (keep it simple!)
‚ùå Low resolution (minimum 300 DPI)
‚ùå Cluttered design (less is more)
‚ùå Poor color contrast
‚ùå Illegible fonts
‚ùå Off-center alignment

## üÜò Troubleshooting

**"My design looks pixelated"**
- Increase resolution to 300 DPI
- Use 4500x5400 pixel dimensions
- Don't resize small images up

**"Text is hard to read"**
- Increase font size
- Use bolder fonts
- Add stroke/outline
- Improve contrast

**"Canva won't let me download transparent"**
- Need Canva Pro for transparent backgrounds
- Alternative: Use remove.bg to remove background
- Or download with white background and specify in Printful

**"AI generated weird text"**
- Don't rely on AI for text rendering
- Generate design without text
- Add text in Canva on top of AI image

**"I'm not creative"**
- Use templates! Search "t-shirt design" on Canva
- Hire a designer on Fiverr ($10)
- Copy successful designs (don't plagiarize, get inspired)
