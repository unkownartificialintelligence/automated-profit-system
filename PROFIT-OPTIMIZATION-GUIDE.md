## Troubleshooting

### Automation Not Running

**Check:**
1. Server is running: `curl http://localhost:3000/api/health`
2. Automation status: Check dashboard
3. Server logs: `tail -f /tmp/server.log`

**Fix:**
```bash
# Restart server
pkill -f "node server.js"
node server.js &
```

### Low Profit Margins

**Check:**
1. Pricing strategy (Settings page)
2. Product costs (Printful)
3. Competition levels

**Fix:**
- Increase prices on trending items
- Focus on high-margin products (mugs, posters)
- Target low-competition keywords

### No Sales

**Common Causes:**
1. Products not published to Printful
2. Payment processing not configured (Stripe)
3. Marketing/SEO needs optimization

**Solutions:**
- Verify Printful API key
- Configure Stripe API
- Enable OpenAI for better product descriptions
- Focus on trending keywords
